Git cheet sheet

ローカルリポジトリをリモートにpushする。

コマンド書式 
git push [remote-name] [branch-name]
branch-nameをremote-nameにpushする。 例：master ブランチの内容を origin サーバー にpushしたい場合 
git push origin master

履歴をたどる

git log: HEAD からたどれるコミットの履歴を見る

リポジトリの新規作成(Githubの場合)

githubにログイン

【New Repository】ボタンにてリポジトリを作成

      mkdir [プロジェクトフォルダ]
      cd [プロジェクトフォルダ]
      git init
      git remote add origin git@github.com:[アカウント名]/[リポジトリ名].git

プロジェクト(リポジトリ)に参加する場合(Githubの場合)

      cd [作業フォルダ]
      git clone git@github.com:[アカウント名]/[リポジトリ名].git

コミットするファイルを指定する

ファイル単位で追加する場合

      git add [ファイル名]
      例：git add hoge.txt

フォルダ単位で追加する場合

      git add [フォルダ名]/.
      例：git add hogedir/.

変更点をコミットする

      git commit -m 'コミット内容'

公開リポジトリに自分のリポジトリの内容を送信する

      git push -u origin master
