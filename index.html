Git cheet sheet

ローカルリポジトリをリモートにpushする。 コマンド書式 
git push [remote-name] [branch-name]
branch-nameをremote-nameにpushする。 例：master ブランチの内容を origin サーバー にpushしたい場合 
git push origin master

履歴をたどる

git log: HEAD からたどれるコミットの履歴を見る

元に戻す

ローカルでの変更をなかったことにする[checkout]

git checkout FILEPATH

ローカルでの変更をなかったことにする[reset]

git reset [--soft / --mixed / --hard]

option repositry index local
--soft 戻る そのまま そのまま
--mixed(default) 戻る そのまま
--hard 戻る 戻る 戻る
